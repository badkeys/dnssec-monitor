# Key Monitoring with badkeys

Various scripts to collect public keys and monitor them for known vulnerabilities with
badkeys.

Currently, DNSSEC is supported, DKIM is planned, more may follow.

You can find summary reports here: <https://monitor.badkeys.info/>
